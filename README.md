For blocks marked with an "*", make sure that there is a period of 
time BETWEEN when skelly gets created and when those code blocks are running.

If skelly is 25 *or* more pixels away from the top left corner then skelly will use a normal
speech bubble too.

This extension is NOT a generative AI or an AI that can talk with people. It is just a 
extension for a sprite with idle animations designed to be made into a companion, tutorial or character.

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://arcade.makecode.com/S27676-31077-47735-64621** and import

## Edit this project

To edit this repository in MakeCode.

* click on **Import** then click on **Import URL**
* paste **https://github.com/jnollen13/skelly** and click import
