time BETWEEN when skelly gets created and when those code blocks are running.

This extension is NOT a generative AI or an AI that can talk with people. It is just a 
extension for a sprite with idle animations designed to be made into a companion or tutorial.

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/jnollen13/skelly** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/jnollen13/skelly** and click import
